## Requisitos

- [x] Postman v10 o superior
- [x] Cuenta en [Postman](https://www.postman.com/)



Cómo importar y ejecutar

### 1. Importar colección y environment

- Abre Postman
- Haz clic en "Import"
- Selecciona ambos archivos ".json" desde las carpetas:
  - /Collections
  - /Environments

### 2. Selecciona el entorno ("Environment")

- En la esquina superior derecha, elige:  
  "PetStore_env"

### 3. Ejecutar colección completa

- Ve a la pestaña "Collections"
- Da clic en la colección y luego en "Run Collection"
